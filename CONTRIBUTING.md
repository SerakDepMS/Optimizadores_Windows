# Guía de Contribución

## Cómo Contribuir

### Reportar Bugs
1. Verifica que no exista ya un issue abierto
2. Usa la plantilla de issue
3. Incluye logs y pasos para reproducir

### Sugerir Mejoras
1. Describe claramente la mejora
2. Explica por qué sería útil
3. Si es posible, propón implementación

### Contribuir con Código
1. Haz fork del repositorio
2. Crea una rama para tu función
3. Haz commit de tus cambios
4. Abre un Pull Request

## Estándares de Código

### Para Archivos .bat:
- Usar encoding UTF-8 con BOM
- Comentar secciones importantes
- Usar nombres descriptivos para etiquetas
- Incluir verificación de administrador

### Para Documentación:
- Escribir en español claro
- Usar Markdown
- Incluir ejemplos prácticos
- Mantener consistencia

## Pruebas
- Probar en Windows 10 y 11
- Verificar con antivirus desactivado
- Probar opciones de restauración
- Documentar cambios en logs
