
# Instalación y Uso - SMS Gaming Optimizer v4.1

## 📥 Instalación

### Método 1: Ejecución Directa
1. Descarga `SMS_Gaming_Optimizer_v4.1.bat`
2. Haz clic derecho sobre el archivo
3. Selecciona "Ejecutar como administrador"
4. Confirma el mensaje de UAC

### Método 2: Desde CMD/PowerShell
```cmd
# Navega a la carpeta del archivo
cd "ruta\a\la\carpeta"

# Ejecuta como administrador
SMS_Gaming_Optimizer_v4.1.bat
```

## 🚀 Primer Uso

### Paso 1: Crear Punto de Restauración
1. Al ejecutar por primera vez, selecciona opción 7
2. Espera a que se cree el punto de restauración
3. Verifica que se haya creado correctamente

### Paso 2: Optimización Básica
1. Selecciona opción 1 (Optimización Rápida)
2. El proceso incluye:
   - Limpieza de temporales
   - Liberación de RAM
   - Optimización de servicios
   - Reparación de sistema
3. Tiempo estimado: 5-10 minutos

### Paso 3: Reinicio
1. **Reinicia tu computadora** después de la optimización
2. Esto aplica todos los cambios correctamente

## 🔧 Configuración Recomendada

### Para SSD:
- Mantener SysMain desactivado
- Usar opción 4 para optimizar servicios

### Para HDD:
- Considerar mantener SysMain activado
- Usar optimización estándar

### Para PCs con poca RAM:
- Usar opción 2 regularmente
- Limpiar temporales semanalmente

## ⚠️ Solución de Problemas

### Error: "Permisos insuficientes"
```cmd
# Solución:
1. Cierra todas las ventanas
2. Ejecuta CMD como administrador
3. Navega a la carpeta del archivo
4. Ejecuta: SMS_Gaming_Optimizer_v4.1.bat
```

### Error: "Antivirus bloquea ejecución"
1. Desactiva temporalmente el antivirus
2. Agrega excepción para el archivo .bat
3. Reactiva el antivirus después de usar

### Error: "No se crea punto de restauración"
1. Ve a Panel de Control > Sistema
2. Configuración de protección del sistema
3. Activa protección para la unidad C:
4. Asigna al menos 5GB de espacio

## 📅 Mantenimiento Regular

### Diario:
- Opción 2: Liberar RAM (antes de jugar)

### Semanal:
- Opción 3: Limpiar temporales
- Opción 6: Ver estado del sistema

### Mensual:
- Opción 1: Optimización completa
- Opción 5: Reparar sistema

## 🔄 Restauración de Cambios
Si experimentas problemas:
1. Usa opción 8 del menú
2. Selecciona qué restaurar
3. Reinicia el sistema
4. Si persiste, usa el punto de restauración creado

## 📞 Soporte
- Revisa el log en `Desktop\Backup_Optimizador`
- Verifica que Windows esté actualizado
- Asegúrate de tener .NET Framework actualizado
